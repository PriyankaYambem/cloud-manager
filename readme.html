About This Simple Cloud Manager

This is a very basic demonstration of a cloud computing web application.

Features:

  • User Registration and Login (securely hashed passwords).
  • Session Management using JWT (JSON Web Tokens) stored in HTTP-only cookies.
  • Protected routes (Dashboard, Read Me) accessible only to authenticated users.
  • Example of user-specific "files" displayed on the dashboard.

How it Works:

The backend is built with Node.js and Express, handling API requests for registration, login, and serving protected pages. The frontend uses plain HTML, CSS, and a little JavaScript to interact with the backend.

Note: The user database is a simple JSON file (users.json) for demonstration purposes. In a real application, you would use a robust database system like MongoDB, PostgreSQL, or MySQL.

To try it:

  1. Register a new account on the home page.
  2. Log in with your new account.
  3. Navigate between Dashboard and Read Me.
  4. Log out to return to the login page.
